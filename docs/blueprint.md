# **App Name**: Foxxage

## Core Features:

- Hero Section: Displays a headline, subheadline, and CTA to capture user attention immediately.
- About Section: Provides an overview of Foxxage, highlighting the benefits of automation for startups and SMEs.
- Services Section: Showcases Foxxage's key offerings—Custom AI Agents, Workflow Automation, Seamless Integrations, and Smart Insights—using visually appealing cards.
- Scroll-Triggered 'Why Choose Us' Section: Utilizes a scroll-triggered animation to smoothly transition between cards, each highlighting a key benefit of choosing Foxxage. Provides subtle feedback in the UI as users explore Foxxage's offering.
- Call-to-Action Section: Encourages users to take the next step by featuring a full-width panel with compelling text and a prominent button to initiate contact.
- Contact Form with Success/Error Messaging: Captures user information (Name, Email, Message) in a sleek form, providing clear success and error messages to improve user experience.

## Style Guidelines:

- Background color: Dark Charcoal (#0D0D0D) to provide a sleek, modern backdrop.
- Primary color: Vibrant Orange (#FF6B35) as the main accent for buttons, highlights, and interactive elements.
- Secondary color: White (#FFFFFF) for text and content to ensure readability, complemented by Cool Gray (#4B4B4B) for secondary text or subtle UI elements.
- Font: 'Inter' (sans-serif) for body text to provide a modern, readable feel. 'Poppins' (sans-serif) to complement, use for headings, sub-headings, and call-to-actions, lending a precise, fashionable touch.
- Utilize flat, minimalist icons that align with the services provided. Orange (#FF6B35) can be used to highlight key icons.
- Implement a grid layout for service cards on desktop, transitioning to a single-column layout on mobile. Maintain consistent spacing throughout all sections.
- Incorporate subtle fade-in effects on page load and smooth hover effects on buttons and cards. Use scroll-triggered card transitions for the 'Why Choose Us' section.
